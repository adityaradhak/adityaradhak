## Featured projects

### 🚗 Auto Insurance Portfolio Analysis
Simulated and analyzed a 10,000-policy personal auto insurance portfolio to evaluate pricing adequacy, risk segmentation, and profitability. Built end-to-end: data generation in Python, analysis in PostgreSQL, and visualization in Excel.  
`Python` `PostgreSQL` `Excel`  
→ [auto-insurance-portfolio-analysis](https://github.com/adityaradhak/auto-insurance-portfolio-analysis)

### 🦠 Actin cytoskeletal network simulation
Research project conducted as part of the UCI MathBioU REU (Summer 2023), supervised by Prof. Christopher Miles. Implemented a continuous-time Gillespie algorithm in MATLAB to simulate competing actin filament elongation and branching dynamics. Ran parameter sweeps across hundreds of trials to identify the optimal elongation/branching rate split (~75%/25%) that minimizes time to reach a target — with implications for whether real cells operate near peak efficiency.  
`MATLAB` `Stochastic simulation` `Mathematical biology`  
→ [actin-network-simulation](https://github.com/adityaradhak/actin-network-simulation)

### 🏠 Bridge the Gap — Dublin Airbnb Market Analysis
Team project submitted to the UCI Atlantis Datathon 2024. Analyzed two weeks of Dublin Airbnb booking data to characterize the market and identify drivers of successful transactions. Built logistic regression and random forest models achieving 81–90% accuracy at predicting booking conversion.   
`Python` `Pandas` `scikit-learn` `Plotly`  
→ [bridge-the-gap](https://github.com/adityaradhak/bridge-the-gap)

### 🎮 Video Game Sales
Exploratory analysis of 5000+ video game titles examining regional sales trends, publisher dominance, and genre preferences. Used linear regression to model Japanese sales and a K-Nearest Neighbors classifier to predict release platform. Completed as a course project for Math 10 at UCI.  
`Python` `Pandas` `Altair` `scikit-learn`   
→ [video-game-sales](https://github.com/adityaradhak/video-game-sales)
